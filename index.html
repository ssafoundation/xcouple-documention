Installation Steps

Node.js

Your computer must have installed nodejs and yarn to run this template. You can download Node.js from https://nodejs.org and yarn from https://yarnpkg.com/lang/en/docs/install/ . NPM comes bundled with Node.js

Yarn

Install Npm Packages

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

Development server

Run npm start for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files.

Build

Run npm run build to build the project. The build artifacts will be stored in the build/ directory. Use the npm run build --prod flag for a production build.

How to change Logo?

It's very easy to change your Image. Just take a look at the screenshot below. Go src > common > Header> Header.jsx and Change the image link to your selected images.


How to Change Background Image

Change Change Background Image::: Go src > components > Home> About> About.jsx and You can change backgroundImage:`url(${aboutBg})`


How to Change PopUp Video Link?

Change PopUp Video Link::: Go src > page > AboutUs> AboutSkill.jsx and You can change Video Link easily ( YouTube, Vimeo etc.)


How to Add facebook Link & Name, Designation?

Add facebook Link & Name, Designation::: Go src > components > Home> Bridesmaids > Bridesmaids.jsx and You can change the following step easily ( Add link & Name)


How to change the counter number?

Go src > page> AboutUs>AboutCounter.jsx and You can change content start= {0}, end= {0}, p= title


How to change the Review Content?

Go src > Page > AboutUs> AboutTestimonials.jsx and You can change content p= Review Content, h2= name
